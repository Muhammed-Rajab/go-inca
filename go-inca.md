### Features 💡

```
1. Set (key, val)
2. Get (key)
3. Set TTL(key, time)
4. Remove TTL(key)
5. Delete (key)
6. Set Eviction Policy (LRU/LFU)
    - LRU:
        - Capacity[MB]
    - LFU
        - Capacity[MB]
```
